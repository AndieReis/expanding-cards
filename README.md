# Expanding Cards Project


## Sobre o projeto:

### O intuito deste mini projeto é aprimorar habilidades em HTML, CSS E JS e treinar conceitos como:

- Animações e transições;
-  Manipulações do DOM;
-  Classes e pseudo classes.
